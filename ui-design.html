Project Overview

This section highlights some of my user interface design work created using Figma, focusing on clean layouts, visual balance, and real-world use cases. These designs demonstrate my ability to craft intuitive and visually appealing interfaces for various industries, including eCommerce, personal branding, restaurants, and photography.

Featured Designs:

  • Vision Glasses Landing Page - A minimal and modern layout designed to showcase a glasses brand with product highlights.
  • Portfolio Website for TheOnlyMaleek - A personalized homepage layout focused on clear hierarchy, clean aesthetics, and professional branding.
  • Seafood Restaurant Hero Section - A bold and visually rich hero section capturing the freshness of the brand with a warm, inviting design.
  • Photography Portfolio Hero Section - A sleek, image-focused hero section meant to highlight creativity and visual storytelling.

Key Features

User-Centered Layouts

Each design was planned with the user journey in mind - emphasizing clarity, simplicity, and engagement.

Visual Hierarchy & Typography

Strong use of font pairing, spacing, and structure to guide users' attention effectively across the interface.

Responsive Design

Though designed primarily for desktop, components were structured with mobile responsiveness in mind.

Built for Real-World Brands

Each concept was inspired by actual business needs and branding principles, making them functional and client-ready.
